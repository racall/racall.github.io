链表学习笔记(一)

开始写之前，小小的抱怨一下，大学里的老师在教C语言时，对于链表这个概念只是提到并没有讲，但个人觉得C语言中的链表知识对于以后的应用还是比较重要的，没办法，老师不讲解，只好自己上网去看视频，查资料学习。

#include <stdio.h>
#include <stdlib.h>
typedef struct sing
{
    int num;
    struct sing *next;
}node;
int main()
{
    node *head = NULL;
    int number;
    do{
    scanf("%d",&number);
    if (number!=-1)
    {
        node *p=(node*)malloc(sizeof(node));
        p->num=number;
        p->next=NULL;
        node *last=head;
        if (last)
        {
            while(last->next)
            {
                last=last->next;
            }
            last->next=p;
        }else{
            head=p;
        }
    }
    }while(number!=-1);
    return 0;
}

上面这段代码，是一个简单的链表的实现，过程是这样的:

1、首先将head赋值为NULL;

2、开辟一个新内存给p;

3、将当前输入的值赋给p->num;

4、把p->next赋值为NULL;

5、把head赋给last
